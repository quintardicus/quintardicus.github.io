## How to Use

### Option 1: View Locally
Simply open **index.html** in your browser to view the portfolio website locally.

### Option 2: Deploy to GitHub Pages
1. Ensure all files are in your GitHub repository
2. Push changes to the main branch
3. Enable GitHub Pages in repository settings (Settings → Pages → Select branch: main)
4. Your site will be live at: `https://[username].github.io/[repository-name]`

## Customization

### Updating Content
1. **Personal Information**: Edit the content directly in [index.html](index.html)
2. **Styling**: Modify CSS variables in [styles.css](styles.css) (lines 2-15) for color scheme changes
3. **Images**: Replace image files with same filenames, or update references in HTML

### CSS Variables (for easy theming)
```css
--primary-color: #0ea5e9;      /* Main blue */
--secondary-color: #6366f1;    /* Purple */
--tertiary-color: #8b5cf6;     /* Violet */
--accent-color: #14b8a6;       /* Teal */
--text-dark: #0f172a;          /* Dark text */
--text-light: #475569;         /* Light text */
```
